Fuente de Laboratorio

A partir de una fuente de alimentación SMPS de 500W obtenida de una computadora, se realiza la modificación para que pueda funcionar como una fuente de laboratorio casera. La fuente provee de voltajes de 3.3V (naranja), 5V (rojo) y 12V (amarillo).

Controlador LED RGB con Arduino

Se diseña un controlador para tiras LED RGB utilizando tres potenciómetros para controlar cada uno de los LEDs y obtener el color deseado. El esquema eléctrico consiste de una placa arduino que emite tres señales PWM con el ancho de pulso proporcional al giro de los potenciómetros. Las señales PWM conmutan tres MOSFET de nivel lógico las cuales se encuentran conectadas con la tira LED.

Además, se realizó una variante en la que se puede utilizar el módulo ESP8266 para controlar mediante WiFi con la librería WLED.

Utilidades

  • Arduino IDE: entorno de desarrollo para las placas Arduino y otros microcontroladores
  • Placa Arduino Uno R3: placa que contiene un microncontrolador, entradas y salidas analógicas/digitales, además de puertos de alimentación e interfaz serial.
  • Tira LED RGB 5050 (tiras analógicas) *
  • MOSFET de nivel lógico **

* Las tiras LED digitales, es decir, programables o direccionables, no funcionan, ya que su control es digital.
** Si no se poseee, requiere acondicionar la señal utilizando un transistor BJT en el gate del MOSFET.

Fuente buck converter basada en Arduino

Se diseña un buck converter en donde se tienen en cuenta los parámetros de frecuencia, pérdidas por conmutación, regulación de voltaje, etc. Dado que en Ecuador es difícil encontrar núcleos de ferrita para realizar los inductores, el buck converter solo existe de forma teórica y con sus respectivas simulaciones.

Utilidades:

  • LTSpice: simulador SPICE de software libre, excelente para electrónica analógica y muchas funciones más (recomendado).
  • Coil32: calcula los parámetros necesarios de vueltas, permeabilidad magnética del núcleo, etc. para obtener la inductancia deseada de un inductor.
  • MATLAB/Simulink: diseño del control y estabilidad del sistema.
